# Алгоритм

## Запуск

Для запуска алгоритма - просто запустить Python файл (должен быть установлен Python). Тогда программа выведет расписание для статичных ручных входных данных.


    python3 main.py

Чтобы запустить программу на случайных тестовых входных данных, закомментировать строки, где вручную создаются groups, rooms, subjects, teachers и раскомментировать предыдущие строки, начиная с faker.Faker() (соответственно, должен быть установлен python faker)
